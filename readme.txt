El ejercicio de realización del modelo de clasificación opté por realizarlo en un jupyternotebook
para hacer uso de markdown.

El codigo puede ser transferido a un archivo py en caso de ser necesario.
Se agrega un url en el archivo 'url_git_collab.txt' que contiene
el link a un collab con permisos de edición para la revisión online del codigo
y el repo en github para ser clonado.

La decisión de usar collab se debe a que programiz no permite instalar las librerias necesarias,
se tomo como alterno el uso de replit pero corrí con el mismo problema.
